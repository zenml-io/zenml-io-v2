## Introduction

This project showcases a simple pipeline on how you can use a local orchestrator and train a Scikit-learn model remotely on VertexAI using a step operator. The artifacts are stored on a Google Cloud Storage bucket and the docker images are stored on Google Container Registry.We will construct a pipeline for this project which loads the data, trains a model, and trains a model remotely in VertexAI.

## Use case

This project structure including the stack and components can be used on occasions you need to construct ML pipelines for tabular or time series data ML problems that require remote infrastructure for training.
